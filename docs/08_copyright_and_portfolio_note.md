# 08 版权与作品集展示说明

本项目用于展示技术能力：

- 小型 RAG 数据建模。
- NPC 人设控制。
- 知识边界与拒答策略。
- Unity 对话气泡交互。
- 本地后端工程组织。

不用于商业发布，不包含官方图像、语音、3D 模型或长台词。

对外展示建议写法：

> Demo characters are used as recognizable test personas for a non-commercial portfolio prototype. Character IP belongs to the respective rights holders. The project focuses on RAG dialogue architecture, local backend integration, and Unity interaction design.

如果担心版权风险，面试投递版可以保留同一套工程，把角色名替换为原创 NPC：

- 阿米娅 → 医疗组织少女负责人。
- 八重神子 → 狐族神社主理人与出版社总编。
- 今汐 → 边境城市年轻执政官。

这样仍能展示同样的工程能力。
