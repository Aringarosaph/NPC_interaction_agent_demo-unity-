# 90 秒录屏脚本

1. 镜头展示白盒场景：三个 NPC 名牌分别是阿米娅、八重神子、今汐。
2. 走近阿米娅，输入：“源石病是什么？”
   - 展示阿米娅短句回应。
   - 打开 debug，显示命中 `amiya_oripathy_infected`。
3. 继续问阿米娅：“你认识八重神子吗？”
   - 展示她不知道，并命中 boundary。
4. 走近八重神子，输入：“我想投稿轻小说。”
   - 展示调侃风格。
5. 走近今汐，输入：“我有一个愿望。”
   - 展示今汐温和回应。
6. 最后展示项目目录和 README：本地后端、数据 schema、Unity 脚本、评测用例。
